## Site Map

* Home: `index.html`
* About Us: `about.html`
* Housing Services: `services.html`
* Loan/Benefits: `benefits.html`
* Documents/Resources: `documents.html`
* Contact Us: `contact.html`

## Page Descriptions

* **Home:** Welcome message, hero/banner, introduction, gallery slider, audio element and quick links.
* **About Us:** Mission, vision, background and services overview.
* **Housing Services:** Three services with images, headings, paragraphs, lists and interactive FAQ.
* **Loan/Benefits:** Three loan/benefit options, comparison table and loan calculator.
* **Documents/Resources:** Downloadable application form and resource links.
* **Contact Us:** Contact form with name, email, phone, service needed and message.

## Tools Used

* HTML5 for page structure
* CSS3 external stylesheet for layout, design and responsiveness
* JavaScript external file for interaction and validation
* Images, SVG graphics and audio media

## JavaScript Features

1. Contact form validation checks required fields, email format and phone format.
2. Dynamic date and time display in the footer.
3. Image slider on the home page.
4. FAQ show/hide section on the services page.
5. Loan payment calculator on the benefits page.
